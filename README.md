# Logistics-Core-C-Based-Sorting-Tracking-System
A comprehensive logistics routing simulator written in C, demonstrating the practical application of core data structures and memory management.
This project simulates a backend system for a delivery network. It parses raw shipment data, filters packages based on priority and storage constraints, and sorts them via an Insertion Sort algorithm. The core engine dynamically routes packages into Linked Lists representing regional Hubs, managing individual Hub workflows using FIFO Queues. Finally, it indexes all active packages into a Hash Table (handling collisions through chaining) to achieve $O(1)$ average time complexity for tracking lookups and delivery finalization.Key Features:Dynamic Memory Allocation: Safe handling of variable-sized datasets without memory leaks.Algorithmic Sorting: Custom implementation of Insertion Sort for priority queuing.Queue & Linked List Management: Multi-layered data structure mapping regional hubs to individual dispatch queues.Hash Table Indexing: Custom string hashing function with chaining collision resolution for rapid record retrieval and deletion.
